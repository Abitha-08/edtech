# 🎓 Integrated Academic & Examination Management System

## 📌 Project Description
A centralized digital platform that automates academic scheduling, exam management, event approvals, hall ticket generation, and seating allocation through role-based access.

---

## 🚀 Features

- 👥 Role-based dashboards (Student, Admin, Seating Manager, Club Coordinator)
- 🎟 Digital Hall Ticket PDF generation
- 🪑 Automated seating allocation (Random algorithm)
- 🎉 Event approval workflow (Pending → Approved)
- 💻 Built using React + Vite

---

## 🛠 Technologies Used

- React JS
- Vite
- JavaScript
- jsPDF
- CSS

## ▶️ How To Run This Project

### Step 1: Clone the Repository

git clone https://github.com/YOUR_USERNAME/edtech.git

### Step 2: Go to Project Folder

cd edtech

### Step 3: Install Dependencies

npm install

### Step 4: Run the Project

npm run dev

### Step 5: Open in Browser

Open the localhost link shown in terminal (usually http://localhost:5173)
